Utilities¶

disaggregator.utils.aggregate_instances(instances, metadata, how='strict')[source]¶

Given a list of temporally aligned instances, aggregate them into a single signal.

disaggregator.utils.aggregate_traces(traces, metadata, how='strict')[source]¶

Given a list of temporally aligned traces, aggregate them into a single signal.

disaggregator.utils.align_instances(instances)[source]¶

Aligns all traces in a list of instances. Removes traces that don’t fit.

disaggregator.utils.align_traces(traces, to=None, how='front', freq=None)[source]¶

Temporally aligns the traces. how`=”front” means to align to the front of the `to trace. If no to trace is given, the first shortest trace is used. Traces are all downsampled to match the lowest sampling rate

disaggregator.utils.bootstrap_appliance_set(appliance_sets, k, n, how='strict')[source]¶

Returns a list of n bootstrapped appliance sets (each with k appliances). Fails if how=”strict” and appliance sets are not aligned.

disaggregator.utils.concatenate_instances(instances, metadata=None, how='strict')¶

Takes a list of instances and concatenates them into a single instance with a single trace

disaggregator.utils.concatenate_traces(traces, metadata=None, how='strict')[source]¶

Given a list of appliance traces, returns a single concatenated trace. With how=”strict” option, must be sampled at the same rate and consecutive, without overlapping datapoints.

disaggregator.utils.concatenate_traces_lists(traces, metadata=None, how='strict')[source]¶

Takes a list of lists of n traces and concatenates them into a single list of n traces.

disaggregator.utils.create_datetimeindex(df)[source]¶

Modify a dataframe inplace to give it a DatetimeIndex. Must have the column ‘time’, which will be removed and used as the index.

disaggregator.utils.generate_random_appliance_sets(appliance_sets, k, n)[source]¶

Given a list of appliance sets, returns n randomly sets, whose instances have been sampled (w/replacement) from the instances of the given appliance_sets.

ApplianceSets must be aligned.

disaggregator.utils.get_common_ids(id_lists)[source]¶

Returns a list of ids common to the supplied lists. (id set intersection)

disaggregator.utils.get_instance_in_time_of_day(device_instance, start_time, end_time)[source]¶

Given an instance and a start and end datetime.time, it returns a trace within that time period.

disaggregator.utils.get_set_in_time_of_day(device_set, start_time, end_time)[source]¶

Resamples all traces in each instance of a given set.

disaggregator.utils.get_trace_in_time_of_day(device_trace, start_time, end_time)[source]¶

Given a trace and a start and end datetime.time, it returns a trace within that time period.

disaggregator.utils.get_trace_windows(trace, window_length, window_step)[source]¶

Returns a numpy array with stacked sliding windows of data from a trace.

disaggregator.utils.get_train_valid_test_indices(n)[source]¶

Given an number n, return three arrays of length n/2, n/4, n/4 respectively, which collectively contain the indices [0..n-1] ordered psuedo-randomly.

disaggregator.utils.get_type_in_time_of_day(device_type, start_time, end_time)[source]¶

Resamples all traces in each instance of a given type.

disaggregator.utils.instances_aligned(instances)[source]¶

Returns True iff instances and their traces are temporally aligned.

disaggregator.utils.order_traces(traces)[source]¶

Given a set of traces, orders them chronologically and catches overlapping traces.

disaggregator.utils.pickle_object(obj, title)[source]¶

Given an object and a filename saves the object in pickled format to the data directory.

disaggregator.utils.resample_instance_traces(device_instance, sample_rate)[source]¶

Resamples all traces within a given instance.

disaggregator.utils.resample_set_traces(device_set, sample_rate)[source]¶

disaggregator.utils.resample_trace(trace, sample_rate)[source]¶

Takes a trace and resamples it to a given sample rate, defined by the offset aliases described in panda time series. http://pandas.pydata.org/pandas-docs/stable/timeseries.html#offset-aliases

disaggregator.utils.resample_type_traces(device_type, sample_rate)[source]¶
