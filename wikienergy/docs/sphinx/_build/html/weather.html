Weather¶

Contains utilities for obtaining weather data and performing temperature normalization. Also includes utilities for converting temperatures to heating/cooling degree days.

disaggregator.weather.degree_day_regression(df, x_opt='both')¶

Function that runs the weather normalization regression on energy use data

df: dataframe that includes
use per day (upd) heating degree days per day (hddpd) cooling degree days per day (cddpd)
x_opt: options for the regression function
‘hdd’: run regression with just heating degree days ‘cdd’: run regression with just cooling degree days ‘both’ (default):
disaggregator.weather.get_cdd(ref_temp, df)¶

Converts a temperature to HDD.

disaggregator.weather.get_hdd(ref_temp, df)¶

Adds a column for heating degree days (converted from temp (F)).
